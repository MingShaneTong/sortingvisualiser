Help
  1. Choose an Algorithm by selecting an option in the dropdown at the top of the screen.
  2. Select the number of items to be sorted.
  3. If you are using the Radix Sort then set the base.
  4. Click the sort button to generate a list of random length.
  5. Click on the list to sort the list from shortest length to longest length. You may also use the space bar once you have deselected the button.
  6. Click the list again or use the space bar to pause and play the animation.
  7. Adjust the Animations slider to change the speed the animations are being shown.
